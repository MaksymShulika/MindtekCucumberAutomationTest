# Mindtek Cucumber Automation Test

This project is a test automation framework built using Selenium, TestNG, and Cucumber.  
It follows the Page Object Model (POM) design pattern to improve code reusability and maintainability.

---

## 🚀 Technologies

- Java
- Selenium WebDriver
- TestNG
- Cucumber (BDD)

---

## 📌 Project Structure

- `pages` → Page Object classes for different web pages
- `steps` → Step definitions for Cucumber scenarios
- `runner` → Test runner configuration
- `utilities` → Driver setup and helper methods

---

## 🧪 Test Coverage

The framework is designed to automate UI testing for different web applications.

It includes:

- Page navigation tests
- UI element validation
- Interaction with buttons, forms, and links
- Multiple page objects for different applications (Etsy, Porsche, WebOrders, etc.)

---

## 🧠 What I Did

- Built a reusable automation framework from scratch
- Applied Page Object Model for better structure
- Used PageFactory to initialize web elements
- Created multiple page classes for different web applications
- Organized test structure using Cucumber and TestNG
